# Guide d'Utilisation - Optimiseur de Streaming Vidéo

## Structure du projet

Organisez vos fichiers comme suit :

```
mon_projet/
│
├── videos.py          # Script principal
├── data/                      # Dossier pour vos fichiers de données
│   └── example.in
│
└── output.txt
```

## Exemple d'execution

'''
python videos.py data/example.in
'''
